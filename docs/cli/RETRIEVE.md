# Vaultify CLI Command - Retrieve

## Overview

The `vaultify retrieve` command allows you to easily pull and unwrap encrypted data from HashiCorp Vault, making it accessible for local use. It simplifies the process of retrieving Terraform state files securely. Into a single command instead of individually executing `vaultify pull` then `vaultify unwrap`.


```bash
vaultify retrieve
```
