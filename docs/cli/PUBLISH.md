# Vaultify CLI Command - Publish

## Overview

The `vaultify publish` command allows you to easily wrap and push the encrypted statefile locally to HashiCorp Vault, making it accessible as a new version in vault or inital version. It simplifies the process of pushing Terraform state files. Into a single command instead of individually executing `vaultify wrap` then `vaultify push`.


```bash
vaultify publish
```
