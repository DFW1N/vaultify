# Vaultify CLI Command - Version

## Overview
The `version` command in the Vaultify CLI allows you to check the version of the Vaultify tool currently installed on your system. It provides information about the tool's version and the platform it is running on.

## Functionality
- **Display Version Information:**
  The `version` command displays the Vaultify tool's logo in ASCII art, followed by the version number. Additionally, it indicates the platform (operating system and architecture) on which Vaultify is running.

## Usage
To check the version of Vaultify installed on your system, run the following command:

```bash
vaultify -v
```
