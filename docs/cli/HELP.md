# Vaultify CLI Command - Help

## Overview
The `help` command in the Vaultify CLI provides a list of available commands and their descriptions, helping users understand how to use the various features of Vaultify.

## Functionality
- **Display Command List:**
  The `help` command displays a list of all available commands in the Vaultify CLI along with brief descriptions of each command.

## Usage
To use the `help` command in the Vaultify CLI, run the following command:
This command displays a list of available commands and their descriptions.

```bash
vaultify -h
```
